# 星球直播

## 直播介绍

鱼皮定期在星球内部直播分享，时间一般是周末，内容包括但不限于：

1. 手把手从 0 到 1 开发完整项目
2. 编程学习经验分享
3. 求职和面试经验分享
4. 编程交流答疑
5. 技术学习心得分享
6. 邀请嘉宾分享

直播过程中，**随时交流互动** ，比如做项目的过程中有问题可以随时抛出。

**所有直播都有超清回放、可下载、无限回看**（支持腾讯会议在线观看、阿里云盘、百度网盘等）。

直播过程中，鱼皮会记录笔记；一个项目完结后，会有完整笔记的整理以及源码下载！遇到什么问题，也可以向鱼皮提问、和其他小伙伴交流、阅读他人的笔记等解决。

此外，鱼皮对外公开直播的回放也可以浏览和下载。
